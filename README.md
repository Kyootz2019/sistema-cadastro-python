# 📇 Sistema de Contatos em Python

Este projeto é um sistema simples de gerenciamento de contatos feito em Python.

## 💡 Funcionalidades
- Adicionar contatos
- Ver lista de contatos
- Buscar contato por nome
- Remover contato
- Editar contato

## 🛠️ Tecnologias
- Python

## 📂 Armazenamento
Os contatos são salvos em um arquivo `.txt` local.

## 🚀 Como executar
Execute o arquivo `main.py`:

```bash
python main.py
